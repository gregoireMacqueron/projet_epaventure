Texte pour le futur mode d'emploi de la version opérationnelle du jeu :

Jeu d'aventure textuel en Python jouable dans le terminal de Windos. 
Les personnages sont créés directement depuis le terminal, sauvegardables et donc persistants.

Comment jouer

Commencez par télécharger les fichiers source au format ZIP à l'aide de l'option disponible dans l'onglet Code ci-dessus. 
Une fois cela fait, extrayez tous les fichiers et utilisez le terminal pour accéder au dossier "xxx" sur votre système. 
Avant de lancer le jeu, installez tous les prérequis à l'aide de la commande suivante : 

$ pip3 install -r requirements.txt

Une fois tout installé, lancer le jeu avce la commande : 

$ python3 xxxxxx.py
